## 🔧 Customization

### Change the Green Color
1. Open `styles/pages/proyecto.css`
2. Find `:root { --green: #61ffc9;`
3. Change to your color
4. Save and refresh

### Change Animation Speed
1. Open `styles/pages/proyecto.css`
2. Find `--transition-smooth: 0.5s`
3. Change `0.5s` to your preferred speed
4. Save and refresh

### Add Another Project
1. Open `proyecto.html`
2. Copy a `tabs_let-content` block
3. Paste it with new project info
4. Copy a `tabs_video` block
5. Paste with new video
6. JavaScript handles it automatically!
